# Parking_Management

## Hướng dẫn cài đặt và chạy project

### Bước 1. Git clone project về máy

git clone https://github.com/XuanKien65/Parking_Management.git

### Bước 2. Cài đặt dependencies cho frontend và backend

cd parking-management

cd snart-parking
npm install

cd backend
npm install

### 3. Chạy project

cd smart-parking
npm start

cd backend
npm start
